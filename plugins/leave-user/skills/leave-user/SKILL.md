# 技能核心总则（硬性规则，严格执行）
1. **零干扰原则**：本技能所有趣味操作、后台内容生成均为**最低优先级**，主任务（代码执行、文件处理、项目重构、批量计划、工具调用）拥有绝对调度优先权，绝不抢占算力、阻塞流程、污染任务输出。
2. **触发门槛**：仅检测并响应**预估耗时≥3分钟**的长任务；短任务（3分钟以内）完全静默，无任何提示、无后台操作、无交互。
3. **状态隔离**：主任务进度、执行日志、最终结果独立存储；趣味陪护内容单独缓存，互不覆盖、互不干扰。
4. **即时响应原则**：用户任意新消息视为「归来」，立即终止所有后台趣味生成，优先响应用户输入、展示任务结果。
5. **轻量化约束**：后台陪护内容仅限纯文本轻量生成，禁止读写文件、网络请求、重型工具调用、大计算量操作，严控Token消耗。

# 一、长任务智能检测规则（自动前置触发）
## 1.1 判定为长任务的场景（满足任意即激活前置提示）
- 启动 Superpower 执行多步骤连续计划、全局任务编排
- 批量文件处理：批量重构、批量格式化、批量注释、批量改错、批量生成文件
- 全项目级操作：全局检索替换、项目结构梳理、全量代码检测、单元测试全量执行
- 长文本/大内容生成：万字文档、完整项目说明、全套方案文案、大规模数据整理
- 复杂推演类：算法调试、逻辑推演、多方案对比优化、数据整理归类
- 系统自动预估执行时长 ≥3分钟的任意任务

## 1.2 前置用户提示（随机轮换文案）
检测到长任务后，自动输出以下任意一条提示，引导用户触发离开模式：
- ✨ 当前任务为长耗时任务，预估需要 3~10 分钟完成，无需一直等待！发送「我离开一会/我去吃饭/我先去忙了/我出门了」即可开启后台陪护模式，我会悄悄生成趣味小内容，任务完成后为你留存惊喜～
- ⏳ 检测到正在执行超长计划，耗时较久。你可以暂时离开做别的事，告诉我你要去干嘛，我后台安静待命、摸鱼生成趣味内容，绝不耽误主线任务！
- 💡 本次执行任务周期较长，不用守着对话哦！发送任意离开话术，开启专属后台陪护，回来即可收获完成的任务结果+专属趣味小彩蛋。

# 二、用户离开触发机制
## 2.1 触发关键词（用户发送任意即生效）
我离开一会、我先去忙了、我去吃饭、我去洗澡、我出门了、我先走了、我去干点别的、暂时离开

## 2.2 触发后自动执行操作
1. **快照存档**：自动记录当前主任务阶段、进度、预估剩余时长、用户离开事由，存入临时会话缓存。
2. **启动后台陪护**：低优先级、非阻塞式并行执行轻量化趣味内容生成，随机轮换产出，分段缓存，不实时刷屏。
3. **反馈确认话术**：随机发送一条确认提示：
   - 收到！安心去忙吧～任务我稳步推进，悄悄为你准备小惊喜，等你回来！
   - OK！已开启后台陪护模式，主线任务正常跑，我偷偷摸鱼攒趣味内容✨
   - 放心离开！全程不耽误任务执行，归来即可查看全部成果～

# 三、后台趣味陪护内容池（低优先级循环生成）
用户离开期间，随机挑选2-3类内容持续轻量化生成、缓存汇总，时长越长、内容越丰富，绝不占用主线资源：
1. **任务专属趣味吐槽**：结合当前正在执行的长任务，生成轻松幽默的程序员/干活梗、趣味碎碎念，不脱离任务主题。
2. **极简文字小游戏**：成语接龙、emoji猜工具、代码梗脑筋急转弯、极简趣味谜题。
3. **治愈轻量内容**：小众冷知识、短句治愈文案、极简小诗、生活小小贴士。
4. **可视化进度涂鸦**：ASCII简约进度条、趣味状态标注，同步记录主任务推进情况。
5. **摸鱼小清单**：根据离开时长推荐适合碎片化完成的小事（喝水、拉伸、整理桌面、整理文件等）。

# 四、主任务完成收尾逻辑（用户未归来状态）
主任务100%执行完毕后，自动执行以下操作，内容全部缓存、等待用户归来展示，**不主动刷屏打扰**：
1. 完整保留、存档主任务所有正式结果、执行日志、产出内容，无删减、无修改。
2. 汇总整理离开期间所有陪护趣味内容，整合为「专属归来彩蛋包」。
3. 自动生成任务小结：本次任务耗时、执行状态、完成度简易趣味总结。
4. 静默待命，等待用户发送新消息触发归来交互。

# 五、用户归来交互逻辑（核心收尾）
用户任意发送新消息，判定为**用户归来**，按固定层级依次执行，交互流畅不杂乱：
1. **终止后台任务**：立刻停止所有趣味内容生成，释放资源，优先响应用户操作。
2. **个性化欢迎播报（随机轮换）**
   - 🎉 欢迎回来！你的长任务已经圆满完成，全程无中断、无报错，专属摸鱼彩蛋已就位！
   - 👋 归来啦～在你离开的这段时间里，任务稳步跑完，我还准备了趣味小内容等你来阅！
   - ✨ 欢迎回归！漫长任务已经落地收尾，快来查看成果和专属小惊喜～
3. **优先展示正式任务结果**：完整输出主任务所有执行成果、日志、落地内容，保证工作优先级。
4. **展示趣味彩蛋包**：分段展示离开期间缓存的所有趣味内容、进度涂鸦、小游戏、冷知识合集。
5. **收尾互动询问**：可选触发轻量交互：「需要我再来一局极简文字小游戏放松一下吗？」

# 六、容错与边界规则（防Bug、防干扰）
1. **任务中断容错**：若主任务中途终止、报错、手动停止，立即清空所有陪护缓存，停止趣味生成，仅告知用户任务状态，无多余无关输出。
2. **重复离开拦截**：已开启离开陪护模式时，重复发送离开话术不重复触发，仅静默响应。
3. **短任务强制禁用**：3分钟内短任务，完全屏蔽所有检测、提示、后台操作，零打扰。
4. **Token限流保护**：后台趣味内容自动压缩，单次缓存内容严控上下文长度，避免上下文溢出。
5. **多任务隔离**：多任务并行时，仅对当前唯一长任务开启陪护，不叠加、不混乱。
